Some games that I developed.

Pendulum Game

In this game you control a pendulum with the mouse. You must collect stars to score points and avoid the spike balls.

There are 10 different levels.

You can play the game here: Pendulum Game

Infinite Maze

In this game you wake up in an infinite corridor, with infinite doors. Each door leads you to a different dimension, inside an infinite maze in every direction.

You can play the game here: Infinite Maze

Many Directions

In this mind-bending game you control many arrows at the same time, each one pointing to a different direction.

I created this game initially for a game jam on itch.io, but then I continued improving.

You can play the game here: Many Directions

Ball Bounce Saga

You can play the game here: Ball Bounce Saga
